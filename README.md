# Prose-App

## Overview
Prose-App is a simple Maven-based Java project for generating prose using imported sentence dependencies.

## Features
- Generate prose from sentences
- Maven build support
- GitHub Actions integration
- Package publishing

## Requirements
- Java 25
- Maven

## Build

```bash
mvn clean install
```

## Usage

Run the application:

```bash
mvn exec:java
```
